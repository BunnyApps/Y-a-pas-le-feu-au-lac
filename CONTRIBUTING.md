# Contribuer à Y'a pas le feu au lac

Votre lac favori ne se trouve pas dans la liste ? Ceux de votre prochain lieu de vacance non plus ?
Alors vous pouvez contribuer au projet.

Pour cela, il existe deux possibilités selon si vous êtes **développeur** ou simple **amateur**.

## Amateur

Il est possible de contribuer au projet en nous envoyant simplement
un mail à l'adresse: teambunnyapps@gmail.com

Les informations nécessaires pour ajouter un nouveau lac sont:

* Le nom du lac
* Pays, département et ville
* Coordonnées (latitude, longitude et altitude)
* Présence d'un parking ou non
* Accès au lac gratuit ou non

## Développeur

Si le développement n'a plus de secrets pour vous, alors vous pouvez contribuez au projet
en suivant les étapes suivantes:

* Faire un fork du projet sur Github
* Créer un branche spécifique pour l'ajout
* Réaliser l'ajout en complétant la class _Lake.java_
* Faire une demande de pull request sur le projet Github
